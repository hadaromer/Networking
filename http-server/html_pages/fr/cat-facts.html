5 Faits Intéressants Sur les Chats

  • Les chats ont cinq doigts sur leurs pattes avant, mais seulement quatre doigts sur leurs pattes arrière.
  • En moyenne, un chat dort environ 12 à 16 heures par jour.
  • Les chats ont une technique de toilettage unique appelée "allogrooming", où ils utilisent leur langue pour nettoyer la fourrure les uns des autres.
  • Le plus ancien chat domestiqué connu a été trouvé dans une tombe vieille de 9 500 ans sur l'île méditerranéenne de Chypre.
  • Les chats utilisent leurs vibrisses pour mesurer les ouvertures et les espaces. Si un chat peut faire passer ses vibrisses à travers un espace, le reste de son corps peut généralement passer aussi.
